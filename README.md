# task-3-Shujat_Ali_Shah
Welcome to the repository for Project 3 of my Virtual Internship as a Data Science and Artificial Intelligence Intern at DecodeLabs. Moving from deterministic rules (Phase 1) and classical classification (Phase 2), this project introduces Recommendation Engines using spatial feature vectors and similarity metrics.

This project implements a terminal-based movie recommendation system that captures user preferences across a high-dimensional genre space and maps them against an item catalog using Cosine Similarity Vector Equations.
